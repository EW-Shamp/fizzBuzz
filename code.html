                            // beginner way to sovle FizzBuzz
                            function renderValues(num, intA, intB) {
                                let arrayData = [];
                                for (i = 0; i <= num.length - 1; i++) {
                                    if (num[i] % intA == 0 && num[i] % intB == 0) {
                                        arrayData.push("FizzBuzz");
                                    } else {
                                        if (num[i] % intA == 0) {
                                            arrayData.push("Fizz");
                                        } else {
                                            if (num[i] % intB == 0) {
                                                arrayData.push("Buzz");
                                            } else {
                                                arrayData.push(num[i]);
                                            }
                                        }
                                    }
                                }
                                return arrayData;
                            }
                            
                            // intermediate way to solve FizzBuzz
                            function renderValuesTwo(num, intA, intB) {
                                let arrayData = [];
                                for (i = 0; i <= num.length; i++) {
                                
                                    Fizz = num[i] % intA == 0;
                                    Buzz = num[i] % intB == 0;       
                                
                                    switch (true)
                                    {
                                        case Fizz && Buzz :{
                                            arrayData.push("FizzBuzz");
                                            break;
                                        }
                                        case Fizz: {
                                            arrayData.push("Fizz");
                                            break;
                                        }
                                        case Buzz: {
                                            arrayData.push("Buzz");
                                            break;
                                        }
                                        default: {
                                            arrayData.push(num[i]);
                                            break;
                                        }
                                    }
                                }
                                return arrayData;
                            }

The code is structered four functions

getValues()

This is the main/controller function of the program. First it retrieves the values from the UI. After-which it parses the values and then validates that they are indeed integers, if not it alerts the user to enter in numbers. Once the values have been validated it calls the generateNumbers() function and stores the returned array into the numbers variable. Next it calls the renderValues() function and passes it the numbers array and both entered integers as parameters and stores the returned array as value. Finally it calls the displayValues() function and passes it value as a parameter.

generateNumbers()

This function creates an array filled with numbers from 1 to 100. I did this as a seperate function to highlight the different ways that is possible to solve the problems in the logic sections.

renderValues()

This function iterates through the array of numbers and checks to see if they are multiples of either integer or both integers. If the number is not a multiple of either integer then it pushes the number to the arrayData array. If the is a multiple of both integers then it pushes “FizzBuzz” to the arrayData array in place of the number. If the number is a multiple of either but not both integers then “Fizz” is pushed to the array if the number is a multiple of the first integer and “Buzz” is pushed to array if the number is a multiple of the second integer.

renderValuesTwo()

This is a more intermediate approach to solve FizzBuzz. Here the function iterates through the passed in num array. It sets the value of Fizz, Buzz, or both true based on if the number in the num array is a multiple of either intA or intB. After-which it performs a switch case to assign the values to arrayData.

renderValuesThree()

This is the more expert way of solving FizzBuzz. The function iterates through the passed in num array. Then uses a ternary operator to determine and assign values based on if the number is a multiple of intA, intB, or both.

displayValues()

This function retrieves the tableBody named “results” and our template named “fbTemplate”. It also clears the tableBody at the start of the function. After the function has retrieved the needed items, it iterates through the array that was passed into the function. It uses the importNode to make a copy of the template and the querySelectorAll to return a static NodeList, after-which it uses classList.add to assign the entry a class and the textContent to assign the entry a value.
